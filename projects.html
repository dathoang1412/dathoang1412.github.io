ii.

GU.AI — AI Fashion Photography Platform

full-stack saas · 2025

Three applications — customer web, a REST API of ~85 endpoints, and an admin panel — letting fashion sellers create product photos and videos through nine AI tools (virtual try-on, model generation, editing) powered by Fashn.ai.

Credit-based billing with PayOS payments, dynamic per-tool pricing and subscription plans; Google Gemini drives an in-studio assistant with RAG-based template search. Real-time notifications and support chat run on Socket.IO, and the admin panel ships live monitoring, RBAC user management and revenue analytics.
